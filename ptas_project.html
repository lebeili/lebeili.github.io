Portable target acquisition system

Year: 2024

Embedded C++ PCB design SolidWorks Control algorithms

The goal of this project was to research and develop a prototype of a portable target acquisition system. The requirements for the prototype given by the company are as follows: the user should be able to use the device to find and designate an arbitrary target in clear weather, and accurately measure the distance between the system and said target (minimum 50 meters). After this, the device should calculate the GPS coordinates of the designated target and display the calculated coordinates for the user. The device should be drone-mountable and therefore weigh no more than one kilogram.
